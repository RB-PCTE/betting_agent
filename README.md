# Personal Odds-Analysis App

## Purpose
This repository contains the Step 1 foundation for a personal odds-analysis application focused on betting research and opportunity detection.

- No automated bet placement.
- Human review is required before any bet.

## Scope (v1)
- Sport: tennis only
- Market: match winner only
- Strategy: Tennis Early Line Drift
- Product mode: alert/research only

- `docs/` — project documentation and implementation plan
